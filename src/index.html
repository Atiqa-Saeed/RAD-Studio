New Features in 11.2

Delphi IOS Simulator

RAD Studio 11.2 now supports the iOS Simulator for Delphi.
You can create simulator apps for macOS devices using ARM-64 chips like M1 and M2.
This makes it easier to test apps on different Apple devices.
Developers can try multiple screen sizes and form factors.
No need to buy actual hardware for testing.
